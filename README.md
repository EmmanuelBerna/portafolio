# Portafolio - Abraham Emmanuel Sanchez Bernabe

Portafolio web personal desarrollado con HTML, CSS y un poco de JavaScript. Presenta informacion personal, proyectos, certificados y datos de contacto.

## Secciones principales

- **Inicio:** presentacion personal y llamada a descargar CV.
- **Slider:** carrusel visual con proyectos actuales y proximos proyectos.
- **Acerca de mi:** trayectoria, habilidades e intereses.
- **Proyectos:** tarjetas con enlaces a repositorios y sitios web.
- **Certificados:** certificados de HTML y CSS de Academia Hola Mundo.
- **Contacto:** enlaces a WhatsApp y correo.

## Tecnologias

- HTML5.
- CSS3.
- JavaScript basico para el slider.
- Imagenes y SVGs locales.

## Estructura

```text
index.html
estilos.css
img/
```

## Como verlo

Abre `index.html` directamente en el navegador.

## Objetivo

Mostrar mi perfil como desarrollador, mis practicas, certificados y formas de contacto en una pagina web sencilla y visual.

## Estado

Proyecto publico y en mejora continua.
